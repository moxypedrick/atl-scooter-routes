Description
Over the course of 2 weeks, I programatically recorded the location of every idle scooter in the Atlanta Area. Using this data I created a database of unique origins and destinations for each scooter at a 5 minute resolution. Using the origins and destinations I determined the best route between the two locaitons using the HERE.com bike routing API. Then I determined how many scooter trips would occur on each road segment in Atlanta. This map shows which roads have head the most scooter use between October 16 and November 1, 2019.
